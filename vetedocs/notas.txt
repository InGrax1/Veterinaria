Página				Función
index_vet.php			Página de inicio de sesión para veterinarios.
home_vet.php			Panel principal tras iniciar sesión.
pacientes_hoy.php		Lista de mascotas con citas del día de hoy, con botón “Atender”.
atender.php5			Página para crear historial clínico desde la cita.
crear_receta.php		Página para generar la receta.
agendar_siguiente.php		Página para agendar la próxima cita del pacien

veterinarios.php		Lista todos los veterinarios registrados (nombre, especialidad, etc.).
registrar_veterinario.php	Formulario para registrar nuevos veterinarios.
editar_veterinario.php		Formulario para editar un veterinario (nombre, especialidad, etc.).
